# AppWebGestionDeArticulos
Aplicación web en ASP.NET WebForms destinada a comercios o emprendimientos que desean exhibir su catálogo de productos a sus clientes. Esta aplicación permite la gestión y visualización de un inventario/lista de productos, con autenticación y área administrativa. 

## Tecnologías
- C#
- ASP.NET WebForms
- SQL Server
- ADO.NET
- Bootstrap
